Realtime communication with WebRTC

Hier finden Sie einige Beispiele für funktionierende Serverstrukturen. Diese sind rudimentär, können aber ohne weiteres um sämtliche gewünschte Funktionen erweitert werden.

Alle Beispiele haben eine Anleitung dabei. Grundlegend bauen alle darauf zuerst einen Server zu starten, anschließend Clients mit diesem zu verbinden und zu guter Letzt Nachrichten zu verschicken oder ein Objekt auf dem Server zu beeinflussen (AuthoritativeServer).
